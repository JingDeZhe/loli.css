Text

Text没什么可说的地方，就是常见的与文字相关的类，诸如text-align与text-transform之类，字体的粗细大小当然也包含于其中，如果想得全面些也可以将text-overflow也加进去，再细当然也可以再加，酌情便好。

.text-ellipsis{
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
.text-clip {
  overflow: hidden;
  text-overflow: clip;
  white-space: nowrap;
}
