HideLine

HideLine主要是隐藏一些不那么重要的话并且在标题或者小元素前加一条竖线以作标志用的，就像是本章的大标题一般，hover的时候隐藏的内容会覆盖标题。

隐藏的内容写在属性hide-data里，:before中通过函数attr()来获取它的值，如果只需要在顶头加一条横线做标志但并不需要隐藏什么值，可以额外添加一个.no-hide类。实现起来就只是:hover的宽度变化和一个transition效果，见代码：

.hideLine{
  position: relative;
  padding-left: 0.5rem;
  &::before{
    content: attr(hide-data);
    padding-left: 0.3rem;
    white-space: nowrap;
    overflow: hidden;
    position: absolute;
    left: 0;
    width: 0.3rem;
    height: 100%;
    transition: 0.2s ease-in-out;
  }
  &:hover::before{
    width: 100%;
  }
  &.no-hide{
    &:hover::before{
      width: 0.3rem;
      transition: none;
    }
  }
