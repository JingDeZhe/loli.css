Card

Card多用来展示图片标签等，同样先看例子，然后便可以看出它可以如何实现。

初看来，并没什么特别之处，貌似什么都可以做到，就算只是单纯地放些文字与图片可以达到这样的效果。仔细一思考，的确是如此的，实际上上面三个.card元素中最右边是假的Card，它就只是元素的顺序排列而已，并没什么高深的样式，这么来说，Card是不是直接就不需要专门写样式了？

如果只需要上面的效果当然是不必要专门写样式了，但是如果换一种样式的话，就能看出来真的.card与假的.card的区别了，我们尝试着让同一行的.card高度一致，注意到.card多数时候是放在.column里的，因为大屏幕中.card的排列与小屏幕中肯定不能相同，不然小屏幕就太挤了，而同行的.column的高度是一致的，所以说想让同行的.card高度一致，只需要将.card的height设为100%就可以，这样一来呈现的便是下边的结果：

很明显可以看出来假.card的区别，它的底部是无法平齐的，而且按照正常的方式它是没法利用传统CSS属性弄成平齐的，所以说假的.card虽然可以，但并不好用，至少页脚块位于最下方它便是无法达到的，这点是常有的样式需求。

说了这么些，其实实现方法也很明了了，便是flex布局，而且并没有任何的技术性可言，直接贴代码：

.card{
  display: flex;
  flex-direction: column;
  box-shadow: 0 0.2rem 0.3rem rgba($dark,0.4);
  border-radius: $border-radius;
  .card-header,
  .card-body,
  .card-footer{
    &:last-child{
      margin-top: auto;
    }
  }
  .card-img{
    width: 100%;
    &:first-child{
      padding-top: 0;
      @include border-radius-top($border-radius);
    }
    &:last-child{
      padding-bottom: 0;
      @include border-radius-bottom($border-radius);
    }
  }
}

如上，将最后一个元素的margin-top设置为auto便可以让它顶到最底端。细节方面，如果.card-img出现在顶部或者底部，就需要设置上下的圆角以适应.card整体的圆角。至于说假.card的样式就不需要给出了，毕竟它实际上没有特别的样式。
